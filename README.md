# Anime.js Letter Animation
A simple web project that animates the letters of a name using HTML, CSS, and the Anime.js library. 

## How to run it
1. Download all three files into the same folder on your computer.
2. Open NO1.html in any web browser to see the animation.

## Files included
* NO1.html - The main webpage and layout
* NO2.css - Styles and colors for the boxes
* NO3.js - The custom Anime.js timeline and rotation rules
